
================================================================================
미디어믹스 시뮬레이터 v2.0 (Media Mix Simulator)
================================================================================

[프로그램 설명]
광고 매체별 예산 배분과 성과를 시뮬레이션하는 웹 애플리케이션입니다.
3가지 시나리오(보수안, 기본안, 공격안)로 성과를 예측하고 Excel 파일로 결과를 저장합니다.

[실행 방법]
1. 필요한 라이브러리 설치:
   pip install -r requirements.txt

2. 웹앱 실행:
   streamlit run app.py
   또는
   run_webapp.bat (Windows - 로컬용)
   
   → 브라우저에서 http://localhost:8501 자동 열림

[팀원 공유 방법]
같은 네트워크(사내망/Wi-Fi)에서 여러 팀원이 함께 사용하려면:

1. run_webapp_share.bat 실행 (Windows - 팀 공유용)
   
2. 실행 시 표시되는 Network URL을 팀원에게 공유
   예: http://192.168.0.10:8501
   
3. 팀원은 공유받은 URL을 브라우저에서 접속

주의사항:
- 방화벽에서 8501 포트 허용 필요
- 같은 네트워크에 연결된 PC만 접속 가능
- 앱을 실행한 PC는 켜져있어야 함

3. 웹앱 모드 선택:
   - 🏠 홈: 대시보드 및 빠른 시작
   - 🤖 AI 자동 예측: 업종별 벤치마크 기반 자동 생성
   - ✏️ 수동 입력: 매체별 데이터 직접 입력
   - 🎯 목표 역산 계산: 목표 전환수/CPA 기반 필요 예산 계산
   - 📁 엑셀 업로드: Excel 템플릿 파일 업로드
   - 📥 템플릿 다운로드: Excel 입력 템플릿 생성

[파일 구조]
madup/
├── app.py                      # Streamlit 웹 애플리케이션 메인 파일
├── requirements.txt            # Python 패키지 의존성
├── run_webapp.bat              # Windows 실행 스크립트 (로컬용)
├── run_webapp_share.bat        # Windows 실행 스크립트 (팀 공유용)
├── README.txt                  # 사용 설명서
├── 미디어믹스_입력.xlsx        # Excel 입력 템플릿 예시
└── saved_presets/              # 프리셋 저장 폴더 (자동 생성)

[사용 흐름]
1. 총 예산 입력
2. 매체 선택 (카테고리별)
3. 각 매체별 상세 데이터 입력:
   - 예산 비중 (%)
   - CPC (클릭당 비용)
   - CTR (클릭률)
   - CVR (전환율)
   - 전환당 매출액
   - 예측 오차 범위
4. 시나리오 조정 폭 설정
5. 계산 및 Excel 저장

[주의사항]
- 예산 비중 합계는 반드시 100%가 되어야 합니다.
- 프리셋 저장 기능을 활용하여 자주 사용하는 설정을 보관할 수 있습니다.
- 저장된 프리셋은 saved_presets 폴더에 저장됩니다.
- 결과 Excel 파일은 브라우저에서 다운로드됩니다.

[출력 파일]
- Excel 결과 파일: 미디어믹스_YYYY년M월_YYYYMMDD_HHMMSS.xlsx
  * 보수안(-5%), 기본안(0%), 공격안(+10%) 시트
  * 시나리오 요약 시트
  * 모든 열 너비 15, 가운데 정렬
  * 퍼센트 형식 적용 (CTR, CVR, ROAS 등)
- Excel 입력 템플릿: 미디어믹스_입력.xlsx
  * 대량 매체 데이터 입력용 (템플릿 다운로드 모드에서 생성)
- 프리셋 파일: saved_presets/[프리셋명].json
  * 자주 사용하는 설정 저장

[주요 기능]
- 🤖 AI 자동 예측: 업종별 벤치마크 기반 자동 미디어믹스 생성
- ✏️ 수동 입력: 실제 캠페인 데이터 기반 정밀 예측
- 🎯 목표 역산: 목표 전환수/CPA 기반 필요 예산 계산
- 📁 Excel 업로드: 템플릿 기반 대량 데이터 처리
- 💾 프리셋 관리: 자주 사용하는 설정 저장/불러오기
- 📊 시각화: 파이차트, 바차트로 결과 시각화
- 💡 AI 인사이트: 시뮬레이션 결과 기반 스마트 추천

[AI 자동 예측 모드]
업종별 벤치마크 데이터를 기반으로 최적의 미디어믹스를 자동 생성합니다.

지원 업종:
- 보험: 네이버, 구글, 메타, 카카오DA
- 패션: 네이버, 구글, 메타, 틱톡
- 금융: 네이버, 구글, 메타
- IT/테크: 네이버, 구글, 메타

자동 적용 요소:
- 업종별 CPC/CTR/CVR 벤치마크
- 계절성 보정 (월별 1~12월)
- 예산 규모 보정 (1천만원 ~ 1억 이상)
- 목표별 예산 배분 (인지도/전환/균형)
- 시나리오 자동 설정 (보수: -5%, 기본: 0%, 공격: +10%)
